2. Installation¶

2.1. Requirements¶

Install JDK 21 or a later version.

Optionally, install rlwrap on Unix like systems. It provides line editing for REPL.

2.2. Install Kink¶

Download a tar.gz file from https://sourceforge.net/projects/kink-lang/files/

Extract kink-{version} directory from the tar.gz file:

$ tar xf kink-{version}.tar.gz

2.3. Install kink.vim¶

If you use Vim to edit Kink programs, consider installing kink.vim
