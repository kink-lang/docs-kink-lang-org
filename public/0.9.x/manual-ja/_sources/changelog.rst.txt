Changed
-------

* In program texts, carriage return (U+000d) can appear only before line feed (U+000a)
* In ``\x{xxxxxx}`` notation of rich :ref:`string tokens <str_token>`,
  surrogate codes can no longer be specified
* $body of :ref:`CONTROL.with_finally <api-a4599f5bf37ae1ee2080db91914c982f>` can no longer
  exit without calling $return.
* Refine the API of :doc:`BIN_HASHER </api/kink-BIN_HASHER>`
* :ref:`PROCESS.exit <api-fba1d75304fad638028d6567d2bd5837>` now only accepts 
  an exit status in [0, 255].
* The exit status from :ref:`Process.wait <api-92659775fbe91963da598772d1f02434>`
  is now unsigned.
* Scan methods of :doc:`input_scanner </api/kink-io-INPUT_SCANNER>` now tail-call $error
  on decoding error.
* Print methods of :doc:`output_printer </api/kink-io-OUTPUT_PRINTER>` now tail-call $error
  on encoding error.
* :ref:`Charset.bin_to_str <api-e15dddbdc2cb0195a47b2dae841f0172>` now raises an exception on
  decoding error.
* :ref:`Charset.str_to_bin <api-590e3a6ff9d9a23d587075b43669a876>` now raises an exception on
  encoding error.
* Rename Str.runes to :ref:`.codepoints <api-486dd39bdf6a9aec769ae7c5f1bd522d>`.
* Rename STR.from_runes to :ref:`.from_codepoints <api-9b2020a08d0489817e2f48a4a900f780>`.
* Rename JAVA.read_only_byte_buffer to :ref:`JAVA.byte_buffer <api-94d74e84a15500dacc000e8516b4c999>`.
* Rename VAL.val_id to :ref:`VAL.id_hash <api-2d36b72d68d811f56ce0ba7352a8818a>`.
* :ref:`Str.repr <api-37a73555aab4b7cde2ae69615d91f7e9>` now produces unpaired surrogates to
  backslash notation.
* :ref:`stderr <api-1d3113732916cec85c49ef707586cb23>` replaces encode error.
* :ref:`stderr <api-1d3113732916cec85c49ef707586cb23>` is synchronized.
* Rename Vec.chunk, Iter.chunk, and IFUN.chunk
  to :ref:`Vec.unconcat <api-aa4ccd1dae957de037c5a4795fcc7df5>`,
  :ref:`Iter.unconcat <api-41349a38ad9d63c669bd8df6ff4dcb15>`,
  and :ref:`IFUN.unconcat <api-1568b429ec04b9367393789d70c86e42>`
* :ref:`IFUN.cycle <api-cb943c256a7d78283fb6008bee85e0e5>` now takes an ifun rather than a thunk.
* Rename Dyn.bind to :ref:`Dyn.with <api-2c27f8f74ce3dab7f33b5d9ca1950786>`.
* Now :ref:`JAVA.char <api-03ee2ace909fd91eacc651a9ad79897a>`,
  :ref:`byte <api-d1e5a7af107a5cb61b33feacd980c2d5>`,
  :ref:`short <api-2fc6b56fcac8415c42d9c704f024fafb>`,
  :ref:`int <api-000436967a852ce467bbb9e721efef6f>`,
  :ref:`long <api-6b26b2272eabd07991c052c23df12077>`,
  and :ref:`big_integer <api-c5e20eff0b1332393ceec0a0ece9e7c9>`
  require that the argument is an integer.
  The first five also validate the lower and the upper boundaries.
* :ref:`Tls.input <api-76f659c5f204bcc5803a95348eb6d612>`,
  :ref:`Tls.output <api-0a2ab2064ccff8b81e4912f65c211dc6>`,
  :ref:`Tcp.input <api-68766f039f7c19637a23ace23fb91f13>`,
  :ref:`Tcp.output <api-ae45015f3f0129c7d60cdd8fa71e61fa>`,
  :ref:`JAVA_CHANNEL_INPUT.new <api-36a6c637d53a52e6f539c9cb961ea1ca>`,
  and :ref:`JAVA_CHANNEL_OUTPUT.new <api-2b4a180ab64898175218543ee74c25cc>`
  now reject Max_buf_size=0.
* Rename IP_ADDRESS to :doc:`IP </api/kink-socket-IP>`.
* Rename IP_ADDRESS_PORT to :doc:`IP_PORT </api/kink-socket-IP_PORT>`.
* Make the parameter of :ref:`Vec.any? <api-11af723e0870efb38f34d5275664b3f9>`,
  :ref:`Iter.any? <api-fe630704531294ad4f5249c4176fa990>`,
  and :ref:`Set.any? <api-edeae5d88aa8bffb3dcd74e23424afb5>` optional.
* Rename Varref.have_val? to :ref:`Varref.any? <api-76b0760cfb86b823ba19b86b27802ba7>`.
* Rename Dyn.have_val? to :ref:`Dyn.any? <api-68457ea90ac062e2c2b824aae11911a8>`.
* Move kink/charset/CHASET to :doc:`/api/kink-CHARSET`.
* Rename Atomic.get to :ref:`Atomic.load <api-b3f8858290eed55096f0c6e4224cbbe8>`.
* Rename Atomic.set to :ref:`Atomic.op_store <api-060528159d8a1a31a749798dea82cea2>`.
* Rename Map.get_maybe to :ref:`Map.get_opt <api-630f4d0db9fbd3d93575a6967dc6b4e2>`
* Inequality operators are now compiled to ``<=`` / ``op_le``, instead of ``<`` / ``op_lt``.
* Rename Set.push_each to :ref:`Set.append <api-81a0643d94216924bffb14257b663f13>`.
* Rename Java.static_type to :ref:`Java.type <api-deb3eecb723aef46d1abb40358fe6b6e>`.
* Rename Java_throw.thrown to :ref:`Java_throw.throwable <api-84973c81a5112a0af9a1dc4f3ceb2b28>`.
* :ref:`JAVA_PROXY.new <api-e983bebce8034ca135e11da64358ffed>`
  now returns a :ref:`java <api-bdcf8aa519e365f452da2436eacd19e7>` value typed as the primary interface.
  Also, proxies provide default implementations for Object methods and interface default methods.
* Now socket options like :ref:`TCP_NODELAY <api-ef70d6f9a0acc6c56641c1359f1b6951>`
  are provided as :ref:`ref <api-a7bb87ccff07365043328fc9f076ecc9>` values.

0.9.4-beta
==========

Added
-----

* :ref:`Iter + Iter <api-3ab44720e374a52836fdd24ac39db2d1>`
* :ref:`IFUN.join <api-174dad688c763676d10f2b6ffa216451>`
* Optional ``From`` and ``To`` parameters
  to :ref:`Vec.iter <api-c31469da21f944429a8cf92a5403ec4f>`.
* Optional ``Min`` parameter to :ref:`Ordered_set.iter <api-2fd2b7ab6519594389ea7aec9583603d>`.
* Optional ``Min_key`` parameter to :ref:`Ordered_map.key_iter <api-33c96f85ef8e3c608fccab2e06618c93>`
  and :ref:`Ordered_map.pair_iter <api-59ee186c19f6fad7a99cdecb2d43e526>`.

Changed
-------

* Generate 32 bytes section id instead of 62 bytes in kinkdoc
* When making an exception with the current traces,
  now the second parameter of :ref:`EXCEPTION.new <api-e8325567ca9201ecedd3460476b31fa1>` can be omitted.
* .reduce methods like :ref:`Vec.reduce <api-2df32705c7bfd54f55ac6f4e6b827746>`
  take an optional fallback continuation for empty streams,
  instead of returning maybe_vec.
* Rename Vec.rev to :ref:`.reverse <api-935ab420e28fb248fb8aae6d385b54a0>`.
* Rename Vec.push_each_xxx methods to append_each_xxx.
* :ref:`Str.search_slice <api-40dc2be7e5fe3839e7c92d0948234e12>` now takes a config fun,
  and returns the index instead of maybe_vec by default.
* :ref:`Regex.match <api-2d4798ecb84f6b0444dc3418bb4960f4>` and :ref:`Regex.search <api-2c5cd9d4f21d040cbf6462c97c5f4897>`
  now take a config fun, and return a match instead of maybe_vec by default.
* :ref:`NUM.parse_int <api-8cd45d8a05be4d520333c05a5c5ad0df>`
  and :ref:`NUM.parse_decimal <api-5f79a184c60f413c3f03d9064c1a8bbd>`
  now take a config gun, and return a num instead of maybe_vec by default.
* :ref:`OFFSET_DATETIME.parse <api-51a47db8ca38fee853999785a20e9213>` now takes a config fun,
  and returns an offset_datetime by default.
* Rename ``on_present`` methods to ``on_success``, and ``on_absent`` methods to ``on_error``.
* Scanner.scan_rune now returns a rune instead of maybe_vec.
* :ref:`Scanner.scan_line <api-24486798989c12e30cd33e07a794967a>` now returns a line instead of maybe_vec.
* :ref:`Input.read <api-e8341e8b849b1926919895693dc3c07e>` now returns a bin instead of maybe_vec.
* Funs and methods now pass an exception instead of an error message to $error: funs of :doc:`FILE mod </api/kink-io-FILE>`,
  :ref:`input <api-0edb5d0790c366443607931bec431fe4>` methods,
  :ref:`output <api-bc8f76dfc800cc663466d8d67cd505ca>` methods,
  :ref:`scanner <api-8d9c7e1d1138bb082067424a205e115b>` methods,
  :ref:`printer <api-d8ea895f044d55406b23b4ecacf9b0ef>` methods,
  :ref:`PROCESS.start <api-382f762afb4d1fd1261301a4d54b00a4>`,
  socket framework, and TLS framework.

Removed
-------

* ITER.chain
* IFUN.chain
* Vec.iter_from
* Vec.search
* Ordered_set.iter_from
* Ordered_map.key_iter_from
* Ordered_map.pair_iter_from
