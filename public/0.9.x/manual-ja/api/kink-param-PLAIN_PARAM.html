6.76. kink/param/PLAIN_PARAM¶

Provides plain_param data type.

  • type plain_param

    • Plain_param.op_store(Arg)

  • PLAIN_PARAM.is?(Val)

6.76.1. type plain_param¶

A plain_param is a param which takes a single argument.

See Vec.op_store for handling of params.

Varref and vec are typical plain_param types.

6.76.1.1. Plain_param.op_store(Arg)¶

Plain_param.op_store consumes Arg as a single argument.

6.76.2. PLAIN_PARAM.is?(Val)¶

PLAIN_PARAM.is? returns whether Val is a plain_param.
