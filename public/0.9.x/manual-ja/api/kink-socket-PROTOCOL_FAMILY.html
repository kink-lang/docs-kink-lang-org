6.91. kink/socket/PROTOCOL_FAMILY¶

ソケットAPIのプロトコルファミリを表す定数を提供する。

  • protocol_family型

  • PROTOCOL_FAMILY.ipv4

  • PROTOCOL_FAMILY.ipv6

  • PROTOCOL_FAMILY.unix

  • PROTOCOL_FAMILY.is?(Val)

6.91.1. protocol_family型¶

protocol_familyの値は、ipv4, ipv6, unixのいずれかである。

protocol_familyの値は相互に、 == 演算子、つまりop_eqメソッドで比較できる。

6.91.2. PROTOCOL_FAMILY.ipv4¶

ipv4は、IPv4プロトコルを表す。

6.91.3. PROTOCOL_FAMILY.ipv6¶

ipv6は、IPv6プロトコルを表す。

6.91.4. PROTOCOL_FAMILY.unix¶

unixは、Unixドメインソケットプロトコルを表す。

6.91.5. PROTOCOL_FAMILY.is?(Val)¶

is?は、Valがprotocol_familyの値かどうかを戻す。
